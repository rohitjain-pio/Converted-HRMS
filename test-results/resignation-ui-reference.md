## Material-UI Components Used

1. **Dialog** - All modal dialogs
2. **Button** - Primary and secondary actions
3. **TextField** - Form inputs
4. **Grid** - Layout for exit details
5. **Typography** - All text content
6. **IconButton** - View icons
7. **Stack** - Button groups
8. **Paper** - Card backgrounds
9. **CircularProgress** - Loading spinners
10. **Tooltip** - Hover information

## Design System Consistency

### Spacing
- Small: 8px
- Medium: 16px
- Large: 24px
- XLarge: 32px

### Typography
- H3: Page titles (28px)
- H4: Section titles (24px)
- Body1: Regular text (16px)
- Body2: Secondary text (14px)
- Caption: Helper text (12px)

### Border Radius
- Small: 4px
- Medium: 8px
- Large: 12px

### Shadows
- Level 1: 0 1px 3px rgba(0,0,0,0.12)
- Level 2: 0 4px 6px rgba(0,0,0,0.12)
- Level 3: 0 10px 20px rgba(0,0,0,0.15)
